# Операция данные удалены
Команда оперативников “Организации” пытается уничтожить группу аномальных
неопознанных существ, расположившихся в комплексе помещений. Помещения состоят из стен,
перегородок и стёкол. Каждый оперативник имеет характеристики и набор снаряжения. Цель
оперативников - уничтожить противников с минимальными потерями.
Команды оперативников и их противников ходят по очереди, при этом любой может ходить и
выполнять действия в ход своей команды. Каждое действие или шаг отнимает определённое кол-во
очков времени у сходившего, которые пополняются каждый раз в начале хода команды. Когда у всех в
команде кончились очки времени, ход передаётся другой команде.
